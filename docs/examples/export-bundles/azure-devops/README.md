# Azure DevOps Export Bundle

Repository-local export generated by Pharabius.

## What This Contains

- `azure-devops-export.example.md` — Example Markdown summary
- `azure-devops-export.example.csv` — Example Azure Boards CSV

## Defaults

- **Work Item Type**: User Story (adjust as needed)
- **Tags**: Semicolon-separated labels

## What This Does Not Do

- Does not call the Azure DevOps REST API
- Does not create work items

PET teams are responsible for review, import, and assignment.

# Jira Export Bundle

Repository-local export generated by Pharabius.

## What This Contains

- `jira-export.example.md` — Example Jira Markdown export (2 drafts)
- `jira-export.example.csv` — Example Jira CSV export

## What This Does Not Do

- Does not call the Jira REST API
- Does not create Jira issues
- Does not assign projects, sprints, or assignees

## How to Use

1. Review the Markdown or CSV content
2. Copy relevant fields into Jira manually or prepare CSV import
3. Add Jira-specific fields (project, sprint, assignee) manually

PET teams are responsible for review, import, and assignment.
